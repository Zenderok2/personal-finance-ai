# Интеллектуальная система управления личными финансами

Этот проект представляет собой интеллектуальную систему для управления личными финансами.

## Функции

- Сканирование квитанций
- Распознавание текста OCR
- Классификация расходов
- Финансовая аналитика
- Рекомендации по бюджету

## Технологии

- Flutter
- Python
- PostgreSQL
- Docker
